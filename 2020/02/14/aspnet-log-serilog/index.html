.NET Core 日志框架：Serilog

发表于 2020-02-14 | 分类于 .NET Core | 阅读次数:

在 .NET Core 日志模型 中对日志整体实现方式进行了介绍，通过使用内置日志记录器来实现日志的输出路径。而在实际项目开发中，使用第三方日志框架来记录日志也是非常多的，首先一般基础的内置日志记录器在第三方日志框架中都有实现，然后很多第三方日志框架在功能上更强大和丰富，能满足我们更多的项目分析和诊断的需求。

本文主要介绍 Serilog 这个日志框架的使用，它是目前比较突出和受欢迎的一个日志框架。Serilog 在日志记录上采用 json 的格式，方便日志的快速查询与过滤。
